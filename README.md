# Blinkit Dashboard Clone

A mini full-stack grocery dashboard inspired by Blinkit/Zomato style apps.

## Features

- Product listing from Express backend
- Add to cart
- Quantity increment and decrement
- Remove item from cart
- Cart total calculation
- Category filter
- Search bar
- Mock payment page
- Payment success page
- Cart persistence with localStorage
- Responsive layout

## Tech Stack

### Frontend

- React
- Vite
- Axios
- React Router DOM

### Backend

- Node.js
- Express
- CORS

## Project Structure

```text
backend/
frontend/
```

## How to Run

- Backend
  cd backend
  npm install
  npm start

- Frontend
  cd frontend
  npm install
  npm run dev
